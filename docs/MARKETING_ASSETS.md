# Marketing Assets - AI Image Generation Prompts

This document contains detailed prompts for generating marketing assets using AI image generation tools (MidJourney, DALL-E, Stable Diffusion, etc.).

## Brand Guidelines

### Color Palette
- **Primary Blue**: `#2C3E50` (Deep professional blue)
- **Gold/Accent**: `#F39C12` (Bright gold for highlights)
- **White/Light**: `#ECF0F1` (Clean white/light gray)

### Design Principles
- **Style**: Modern, minimalist, professional yet approachable
- **Typography**: Modern sans-serif, clean and readable
- **Aesthetic**: Educational but playful, sophisticated yet accessible
- **Focus**: Keep designs minimal and focused, avoid clutter

## 1. App Icon (1024x1024px)

### Requirements
- Format: Square 1024x1024px, PNG with transparency
- Must be recognizable at 60x60px (smallest size)
- No text or letters
- iOS app icon style with rounded corners

### Prompt

```
Create a modern, minimalist app icon for a chess learning mobile game.

Visual Elements:
- Feature: Stylized chess pieces (king and knight) in bold silhouette
- Composition: King piece slightly forward, knight piece behind or beside
- Style: Flat design with clean, smooth lines
- Detail level: Simple but distinctive, recognizable at small sizes

Color Scheme:
- Background: Deep blue (#2C3E50)
- Pieces: Gold/white gradient (#F39C12 to #FFFFFF)
- Optional: Subtle shadow or depth effect for pieces

Design Elements:
- Include subtle upward arrow or progress indicator integrated into the design
- Could be part of the crown on king, or negative space
- Conveys: Growth, learning, improvement

Aesthetic:
- Professional but not intimidating
- Educational yet playful
- Sophisticated but accessible
- Modern and current (2025 design trends)

Technical Requirements:
- Square format with rounded corners (iOS style)
- High contrast for visibility
- Clean silhouettes
- Must work on both light and dark backgrounds
- No gradients in background (solid color preferred)
- Pieces should have clear, recognizable chess piece shapes

Avoid:
- Photorealistic textures
- Complex patterns
- Small details that won't scale
- Text or letters
- Multiple colors (keep it 2-3 colors max)
```

### Variations to Generate
1. **Primary**: King + Knight composition
2. **Alternate 1**: Solo king with progress arrow
3. **Alternate 2**: Knight piece with upward movement lines
4. **Alternate 3**: Abstract chess board pattern with piece silhouette

## Technical Specifications

### File Formats
- **App Icon**: PNG with transparency, 1024x1024px
- **Screenshots**: PNG, device-specific dimensions
- **Graphics**: PNG or JPEG, high resolution (2x target size minimum)

### Resolution Requirements
- All assets: Minimum 2x resolution of target size
- App icon: Exactly 1024x1024px
- Screenshots: Native device resolution
- Web graphics: 150-300 DPI for print quality

## AI Tool Recommendations

### MidJourney (Best for artistic style)
- Excellent for app icons and illustrations
- Add parameters: `--ar 1:1` for square, `--quality 2` for higher quality
- Style modifiers: `clean`, `minimalist`, `modern`, `professional`

### DALL-E 3 (Best for UI mockups)
- Great for screenshot mockups and UI elements
- Specify iPhone mockup for realistic device frames
- Good at following detailed compositional instructions

### Stable Diffusion (Best for customization)
- Full control over generation
- Use with ControlNet for precise layouts
- Models: SD XL for high resolution

### Figma + AI Plugins (Best for actual screenshots)
- Use Figma to design actual screens
- Export as high-res images
- Most accurate representation of app
